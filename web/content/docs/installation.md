# Installation

## Prerequisites
- Node.js 18+

## Install
```bash
npm install -g mcpm-dev
```

## Verify
```bash
mcpm-dev --version
```

## Platform Notes

### macOS
```bash
brew install node
npm install -g mcpm-dev
```
