# API Reference

Base URL: `https://www.mcpm.dev/api`

## Packages

```bash
GET /api/packages
GET /api/packages?q=weather
POST /api/packages
DELETE /api/packages/:name
```

## Content

```bash
GET /api/content?type=blog
POST /api/content
```
