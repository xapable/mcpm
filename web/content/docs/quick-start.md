# Quick Start

## 1. Install
```bash
npm install -g mcpm-dev
```

## 2. Login
```bash
mcpm-dev login
```

## 3. Create a Tool

Create `package.json`:
```json
{"name":"hello-mcp","version":"1.0.0","main":"server.js"}
```

## 4. Publish
```bash
mcpm-dev publish
```
